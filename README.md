# Hotel Booking System

A comprehensive hotel booking management system that allows users to book rooms, manage reservations, and handle hotel operations efficiently.

## Features

- Room Booking Management
- Room Categories and Pricing
- Reservation Status Tracking
- Admin Dashboard
- User Authentication and Authorization

## System Requirements

- PHP
- MySQL

## Usage

### User Features
- Register/Login to the system
- Browse available rooms and their details
- Make room reservations
- View booking history
- Cancel or modify bookings

### Admin Features
- Manage room inventory
- Handle reservations
- View booking statistics
- Update room pricing and availability
